しくみ / HOW IT WORKS

1. パレット＝CSSトークン一式 — 全部品が var(--royal) 等を参照。<html data-palette="gameboy"> を書き換えるだけで全画面が即・着せ替え（このページと同じ）。
2. ユーザーごとに保存 — User.palette カラムを1つ追加。マイページで選択 → Server Action で保存 → layout.tsx がSSRで data-palette を刻むから、リロードしてもチラつかない。
3. 増やすのが簡単 — 新パレット＝CSS十数行。季節限定（SAKURA、HALLOWEEN…）やレベル到達で解禁される“ごほうびパレット”にも展開できる。
